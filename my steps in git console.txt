git version - проверяю версию продукта
git status - проверяю статус (на этом этапе нашлись 2 модифицированных файла, позже мы добавим изменения отдельной командой)
git clear - очистили терминал
git branch olya - создаем новую ветку под название "olya"
git checkout olya - переключаемся на новую ветку "olya"
git branch -  проверяем, перешли ли мы на новую ветку "olya"
git checkout -b fake - создаем новую ветку с мгновенным переключением на нее
git add my `.step in git console.txt` - хотим добавить изменения в систему
git add . - добавить все изменения в систему
git help - подсказки по командам
git commit -m `.my steps in git console.txt` коммитим изменения
git push - отправляем правки на сервер
git merge olya - соединить изменения ветки, в которой находимся мы с той веткокой, которую мы выберем (olya)
